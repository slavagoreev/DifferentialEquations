Solution of a differential equation

We were given an equation $y'=-2y+4x$ and inital values $x_0=0$, $y_0=0$ and $X=3$ This is a first order linear ODE has the form of $y'\left(x\right)+p\left(x\right)y=q\left(x\right)$. Using an integration factor we may find the general solution in form: $y\left(x\right)=\frac{\int \:e^{\int \:p\left(x\right)dx}q\left(x\right)dx+C}{e^{\int \:p\left(x\right)dx}}$ As we can see, $p\left(x\right)=2,\ q\left(x\right)=4x$, consequently the integrating factor is: $\mu \left( x \right) = e^{\int {p\left( x \right)dx} } =e^{2x}$ Multiply by the integration factor, $\mu(x)$ and rewrite the equation as: $e^{2x}y'\:+e^{2x}\cdot \:2y=e^{2x}\cdot \:4x$ $\left(e^{2x}y\right)'\:=e^{2x}\cdot \:4x⇒e^{2x}y=\int \:e^{2x}\cdot \:4xdx⇒e^{2x}y=2e^{2x}x-e^{2x}+C$ If we divide both parts by $e^{2x}$ then we may isolate $y$: $y=\frac{2xe^{2x}-e^{2x}+C}{e^{2x}}$ Substituting initial values $x_0=0$, $y_0=0$, we may find out $C = 1$
