We were given an equation $y'=xy^2-3xy$ and inital values $x_0=0$, $y_0=2$ and $X=6.4$

This is a first order separable ODE that has a general solution in from of: $y=-\frac{3}{e^{C+\frac{3x^2}{2}}-1}$

Let's build an approximation using:

  1. Euler's method
  2. Improved Euler's method
  3. Runge-Kutta method
